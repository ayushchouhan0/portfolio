Student Segmentation and Course Recommender Engine

Scikit-Learn Pandas NumPy Machine Learning Clustering Data Analysis Feature Engineering
Live Frontend Invoke Backend View Code

Project Overview

Built an intelligent customer clustering and course recommendation system using Agglomerative Hierarchical Clustering and Random Forest Classifier to segment students based on demographic, psychometrical, and academic attributes. This enables targeted course recommendations and personalized marketing strategies that drive enrollment and engagement.

Key Features

  • Advanced Clustering: Agglomerative Hierarchical Clustering for natural, interpretable grouping
  • Multi-dimensional Analysis: Analyzes 20+ demographic, behavioral, and academic attributes
  • Personalized Recommendations: AI-driven course suggestions based on cluster characteristics
  • Marketing Optimization: Targeted campaign strategies for different student segments
  • Predictive Analytics: Suitable course recommendation prediction based on user features
  • Interactive Dashboard: Real-time Prediction visualization and insights

Technical Implementation

Utilized scikit-learn's advanced clustering algorithm and Ensemble learning package combined with pandas and numpy for comprehensive data preprocessing and feature engineering. The system processes multiple data dimensions including academic performance, learning preferences, engagement patterns, demographics, and behavioral indicators to create actionable student segments.

Business Impact & ROI

The segmentation and recommender engine improved course enrollment rates through personalized recommendations and increased marketing campaign effectiveness. The system now processes 500+ customer profiles monthly, helps counsellors for product pitching.
