Mental Health Assessment Chatbot

Live Frontend Invoke Backend View Code

Project Overview

Developed an intelligent RAG-based conversational AI system that provides mental health support and conducts comprehensive assessments with audio response capabilities. The system combines contextual chatbot interactions with structured mental health evaluations to deliver personalized insights and support recommendations.

Key Features

  • RAG-Powered Conversations: Contextual responses using pre-built mental health knowledge base
  • Intelligent Assessment Triggers: Automatic mental health evaluation suggestions after meaningful interactions
  • Audio Response Processing: Real-time speech-to-text conversion for assessment answers
  • Comprehensive Report Generation: AI-generated insights combining chat history and assessment data
  • Session Management: Maintains conversation context and user interaction history
  • RESTful API Architecture: Clean, documented endpoints for seamless integration
  • Multi-modal Interaction: Supports both text and audio input for accessibility

Technical Implementation

Built using FastAPI framework with Groq Llama-3.3-70b-versatile for natural language processing and HuggingFace sentence-transformers for embeddings. Implemented FAISS vector store for efficient similarity search and Groq Whisper-large-v3 for audio processing. The system processes 7-question structured assessments with real-time audio transcription and generates detailed mental health reports using advanced prompt engineering.

Business Impact & ROI

The mental health chatbot system provides 24/7 accessible mental health support with personalized assessments and recommendations. The solution enables healthcare providers to conduct preliminary screenings efficiently while maintaining conversation context for meaningful therapeutic interactions. Currently supports multiple concurrent users with real-time processing capabilities.
