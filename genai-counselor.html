Project Overview

Developed an intelligent counseling chatbot using multiple Large Language Models (OpenAI, Llama, Gemma, and Anthropic) integrated through Ollama and Hugging Face. The system implements Retrieval-Augmented Generation (RAG) architecture to provide context-specific responses for educational counseling.

Key Features

  • Multi-LLM Integration: Leverages different AI models for optimal response generation
  • RAG Architecture: Implements dynamic retrieval and response generation through vectorization of course data
  • Cloud Integration: Utilizes GCP cloud storage for scalable data management
  • API Deployment: FastAPI-based deployment for seamless integration
  • Database Integration: Connected with course databases for real-time information

Technical Implementation

The chatbot uses FAISS (Facebook AI Similarity Search) for efficient vector similarity search, enabling quick retrieval of relevant course information and frequently asked questions. The system processes course overviews and QnA data through advanced vectorization techniques, ensuring contextually relevant responses.

Impact & Results

This project significantly improved student engagement and provided 24/7 automated counseling support, reducing response time from hours to seconds while maintaining high accuracy in course recommendations. The system handles over 1000+ queries daily with 95% satisfaction rate.

Note: The deployed functional version of the project is a RAG based chatbot which supports user uploaded documents only to be used for context retrieval.
